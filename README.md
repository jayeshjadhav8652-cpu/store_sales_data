# Store Sales Dashboard (Excel)

## Project Overview
Developed an interactive Store Sales Dashboard using Microsoft Excel to analyze sales performance across cities, regions, years, customer segments, and product categories.

## Key Insights
- City-wise Sales, Profit and Quantity Analysis
- Region-wise Sales Distribution
- Year-wise Sales and Profit Trends
- Customer Segment Analysis (Consumer & Corporate)
- Product Category Filtering
- State-wise Sales Analysis

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Dashboard Design

## Dashboard Features
✔ Interactive Slicers

✔ Dynamic Pivot Charts

✔ Sales Trend Analysis

✔ Profitability Tracking

✔ Regional Performance Monitoring
